# Nest Store Documentation

Comprehensive documentation for the Nest Store E-commerce platform. This project contains the design specifications, API documentation, and technical guides for the system.

## Project Overview

Nest Store is a modern, scalable e-commerce solution designed for high stability and complex business workflows.

### Core Technology Stack

- **Backend**: Java 21 with Spring Boot 3.4.5
- **Database**: PostgreSQL
- **Workflow & Messaging**: Temporal, Apache Kafka
- **Security**: Spring Security, OAuth2, Casbin
- **Documentation**: Docusaurus V3 with TailwindCSS & Shadcn/UI

## Project Structure

- `docs/`: Technical documentation and system design.
- `blog/`: Project updates and technical articles.
- `api-swagger/`: OpenAPI specifications for the system services.
- `src/`: Custom documentation components and pages.

## Getting Started

### Local Development

1. **Install Dependencies**:

   ```bash
   npm install
   ```

2. **Start Development Server**:

   ```bash
   npm start
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Deployment

The documentation is configured for deployment via Vercel or other static site hosting platforms.
