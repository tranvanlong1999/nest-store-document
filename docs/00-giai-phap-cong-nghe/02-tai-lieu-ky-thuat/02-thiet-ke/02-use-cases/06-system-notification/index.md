# System Notification

Handles alerts and internationalized messages.

## Notifications

User-specific notifications.

**Schema:**
```sql
create table notification
(
    uuid_notify varchar(40) not null
        primary key,
    user_id     bigint, -- Note: Type doesn't match uuid_user varchar(40) in other tables, check consistency
    type        varchar(15)
);
```

## Messages (I18n)

System messages and translations.

**Schema:**
```sql
create table message
(
    id      integer generated by default as identity
        primary key,
    locale  varchar(255),
    key     varchar(255),
    content varchar(255)
);
```
